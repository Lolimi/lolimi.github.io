About RCHoppers

RCHoppers started out as my first real spigot plugin project on a real SkyBlock server with other people: the RCCrafters server, that's also where the name comes from. What started out as a performance heavy plugin with just a few functions, became a huge project, that helps a lot on the performance and adds lots of features.
After the Chunk Hopper, that picks up items in an entire chunk and saves on the items physics calculations, came the Sorting Hopper, that just helps with your sorting systems and reduces the amount of hoppers and redstone needed.
The Storage Unit Hopper helps reducing the amount of chests needed, if you have a farm, that produces a lot of one item type: it allows you to store an unlimited amount of one item type in only one block, respecting different item names and lores.
To make the sorting system perfect there is also a hopper for your overflow: The Void Hopper, that just voids any item that get's inside of it.
The newest addition is the Auto Crafter: The only item, that is not a hopper. But it does exactly what you would think: it autocrafts items. The player can set the item to be autocrafted in a setup gui and then the items can be put in using hoppers and taken out using another hopper below the crafter.

Since this plugin was highly custom for the servers needs and had everything hard coded, there was no way it could be published. So i decided to give it a complete overhaul, adding new ways to make even easyer on the server performance and also more customizable. In the mean time i also got interested in learning html, so i decided to write my own website, which you are on at the moment.
The real big thing about the new RCHoppers is that it is easy to make plugins for it, using the main RCHoppers plugin. So if you are a developer and want to try something new and relatively simple you are very welcome to contact me, your project could then also be uploaded here!
